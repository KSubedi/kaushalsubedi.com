Relix Timer Pro

Relix Timer Pro is an Android that tracks past and future events.

Relix Event Timer Pro is a minimalist and lightweight timer application that will let you create countdown and count-up timers that show you exactly how many years, months, weeks, days, hours, minutes and seconds have passed since your event or need to pass till an event. Relix Event Timer Pro is soon going to be open sourced!
