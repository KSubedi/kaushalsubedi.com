Nepali Text To Speech

Nepali TTS helps you convert any language to Nepali text and speech.

Nepali Text To Speech is a tool developed within 3 hours after the big earthquake of Nepal in 2015 so that the international rescue workers could communicate with the locals easily. It takes any language input and outputs Nepali written text as well as speech. It has been used extensively by International Red Cross and other organizations working in Nepal.
