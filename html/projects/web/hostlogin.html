Host Login

Host Login helps you organize your web hosting details.

Host Login was created to help System Administrators and others organize their web hosting login and other information in an all in one dashboard. The contents are encrypted end to end and cannot be decrypted without explicitly entering the password used. This project was originally created to be used as a web service, but has been open sourced now.
